El elemento HTML define una sección que se cita de otra fuente.
Los navegadores suelen aplicar sangría a los elementos.

La etiqueta HTML define una cita corta. Los navegadores normalmente insertan comillas alrededor de la cita.

La etiqueta HTML define una abreviatura o un acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Marcar abreviaturas puede proporcionar información útil a los navegadores, sistemas de traducción y motores de búsqueda. Propina: Utilice el atributo de título global para mostrar la descripción de la abreviatura/acrónimo cuando pase el ratón por encima del elemento.

La etiqueta HTML define la información de contacto del autor/propietario de un documento o un artículo.

Written by John Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

La etiqueta HTML define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).

The Scream by Edvard Munch. Painted in 1893.
