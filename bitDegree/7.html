funcion map

La función de mapa de JavaScript se aplica cuando es necesario llamar a una función para cada elemento de una matriz.
A partir de los resultados generados, el método creará una nueva matriz.
Por lo tanto, la primera matriz permanecerá sin cambios: todas las modificaciones de los elementos se representarán en la nueva matriz .
Si aplica la función de asignación de JavaScript, debe usar la nueva matriz para que el uso de este método valga la pena
